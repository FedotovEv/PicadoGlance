Открытие (загрузка) файлов.

Прежде всего, среди всех инструментов программы следует выделить средства загрузки PCAD-файлов. Они осуществляются в программе с помощью специальных автономных модулей - загрузчиков. Для открытия файла можно применить либо все доступные загрузчики сразу (пункт меню "Файл/Открыть"), либо какой-либо один определённый загрузчик (пункты подменю "Файл/Открыть как"). Для предварительного определения набора апертур, которые необходимы для загрузки многих типов поддерживаемых файлов, следует воспользоваться средствами настройки программы.

Просмотр изображения.

Просмотр загруженного файла выполняется в основном окне. В нём может быть показана только часть всего изображения, а для возможности просмотра полной площади картинки программа предоставляет инструменты двух типов - инструменты перемещения окна просмотра по поверхности загруженного изображения, а также инструменты изменения масштаба этого изображения.

Описание инструментов изменения масштабов можно найти по ссылке выше. Что же касается способов перемещения окна просмотра, то они существуют следующие: полосы прокрутки по сторонам основного окна, а также различные клавиши клавиатуры и их сочетания. Две ближайшие к окну просмотра полосы прокрутки предназначены для перемещения его по площади текущей картинки. Работают они вполне интуитивно - вертикальная полоса прокрутки (прилегает к окну просмотра справа) смещает окно по вертикали соответственно направлению движения её лифта, а горизонтальная полоса прокрутки (прилегает к окну просмотра снизу) аналогично двигает окно просмотра по горизонтали. Величины лифтов этих полос символически отображают текущий размер окна просмотра (зависит от установленных масштабов) относительно полного размера активного изображения.

Двигать окно просмотра можно также посредством клавиш управления курсором. Чтобы использовать этот способ, окно просмотра должно иметь фокус ввода. Поддерживаются четыре различных скорости движения окна. Нажатие какой-либо из "стрелочных" клавиш управления курсором перемещает окно в соответствующем направлении на самой малой скорости. Нажатие и удержание клавиши Ctrl одновременно со "стрелочными" клавишами обеспечивает вторую (вчетверо более высокую) скорость движения окна. Клавиши PgUp, PgDn двигают окно по вертикали на третьей скорости (в двадцать раз выше самой малой), клавиши Home и End действуют аналогично, но уже применительно к смещению окна по горизонтали. Наконец, эти четыре клавиши в сочетании с удержанием кнопки Ctrl вызывают соответственное перемещение окна на самой высокой, четвёртой скорости - в сто раз выше минимальной первой.

Экспорт изображения.

Одна из основных функций программы - конвертирование изображений в ряд других типовых общераспространённых графических форматов. Эта операция выполняется посредством меню "Файл/Экспорт". Через это меню можно указать желаемое подмножество примитивов для экспорта, а также указать требуемый целевой графический формат. Более детальная информация о процедуре выполнении экспорта находится здесь.

Печать.

Очень похоже на экспорт выполняется и операция печати. Подменю "Файл/Печать" содержит набор пунктов, позволяющих выбрать совокупность распечатываемых элементов, а также пункт для непосредственного выполнения печати.

Селекция и копирование примитивов.

Одним из инструментов выделения примитивов является их отметка. Управлять совокупностью отмеченных примитивов можно пунктами меню "Правка", а также группой из трёх кнопок на полосе инструментов главного окна. Более детальное описание назначения и приёмов работы с ними можно найти здесь. Полученную в результате группу отмеченных элементов можно распечатать, направить на экспорт, а также скопировать в буфер обмена операционной системы. Последняя из перечисленных операций выполняется пунктом меню "Правка/Копировать".

Для направления же меченых примитивов на печать или экспорт следует активировать пункт "Отмеченное" соответствующего меню (меню "Файл/Печать/Отмеченное" для распечатки и "Файл/Экспорт/Отмеченное" для экспорта). В обоих этих случаях данный пункт представляет собой двоичный переключатель, который можно перебросить в противоположное состояние его выбором мышью или клавиатурой. Активность режима указывает флажок слева от названия пункта меню.

Операции управления видом отображаемого изображения.

Операции определения вида изображения сведены в единое подменю "Вид". Его пункты позволяют установить желаемый масштаб изображения, редактировать его слоистую структуру, задавать требуемые шрифты для рисовки текстовых примитивов картинки, а также выбрать один из поддерживаемых языков интерфейса программы, для которого существуют и установлены языковые ресурсы (перевод).

Инструмент "Холст" и управление им.

Подменю "Холст" объединяет в себе операции управления особым инструментом размещения и выделения примитивов - - холстом. Пункты этого меню позволяют выбрать размер (стандартный или нестандартный), ориентацию (портретную или ландшафтную), а также включить либо выключить его. Нестандартные (произвольные) размеры, ориентация и активность холста могут быть также изменены вертунами и кнопками-переключателями полосы инструментов главного окна программы.

Установка нестандартного размера холста выполняется с помощью пункта "Холст/Установка размера", а его последующий выбор в качестве рабочего - пунктом меню "Хост/Тип/Переменный". Выбор пункта "Холст/Установка размера" вызовет на экран следующий диалог.

Конфигурация программы.

На данный момент настройка программы осуществляется через один диалог параметров программы. Он позволяет выбрать общие настройки, к числу которых относится набор используемых апертур (засветок), используемые при вводе/выводе единицы измерения размеров, применение и величина дополнительного масштабного коэффициента и необходимость применения подмасштабирования при рисовании на мониторе (более детальную информацию о последних двух параметрах можно получить здесь).

Система помощи по работе с программой.

Наконец, последний раздел главного меню "Помощь" и последняя кнопка инструментария вызывают и активизируют данную систему помощи, с которой вы сейчас и работаете. Пункт подменю "Помощь/О программе" вызывает на экран окно с краткой общей информацией о программе.
